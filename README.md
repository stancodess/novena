# Novena de Aguinaldos 2025 - Día Sexto

## Estructura del Proyecto

```
NOVENA 2025/
├── novena.html          # Archivo principal HTML
├── images/              # Carpeta para imágenes
│   └── pesebre.png     # Imagen del pesebre
└── videos/              # Carpeta para videos
    └── video_navidad.mp4 # Video de la canción navideña
```

## Instrucciones

### Imágenes
- Coloca todas las imágenes en la carpeta `images/`
- Formato recomendado: PNG o JPG
- La imagen del pesebre ya está incluida: `images/pesebre.png`

### Videos
- Coloca todos los videos en la carpeta `videos/`
- Formato recomendado: MP4
- El video de la canción ya está incluido: `videos/video_navidad.mp4`

## Notas
- Asegúrate de mantener la estructura de carpetas para que las rutas en el HTML funcionen correctamente
- Todas las rutas en el HTML están configuradas para usar las carpetas `images/` y `videos/`
